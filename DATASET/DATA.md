## ImageNet-1K Preparation

We use standard ImageNet dataset, you can download it from http://image-net.org/. Move the validation set uner `DATASET`.
  ```bash
  DATASET
   ├──DATA.md
   └──imagenet
        └── val
            ├── meta.json
            ├── n01440764
            │   ├── ILSVRC2012_val_00000293.JPEG
            │   └── ...
            └── n02102318
            └── ...
  ```
